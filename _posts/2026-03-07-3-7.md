<h2>BackTracking</h2>
<h3>설명</h3>

<h3>해시 집합</h3>
<p>중복을 허용하지 않는 고유한 요소들의 집합을 관리하기 위해 해시 테이블을 기반으로 구현한 자료구조이다.</p>
<p>해시 함수를 통해 데이터의 저장 위치를 결정하므로 검색, 삽입, 삭제 평균 O(1)의 빠른 시간 복잡도로 수행 할 수 있어</p>
<p>데이터의 유무 확인에 매우 효율적이다.</p>

<pre><code>def find_duplicates_hash(nums):
    """
    방법3: 해시 집합 사용
    시간 복잡도: O(n)
    공간 복잡도: O(n)
    """
    seen = set()
    duplicates = set()

    # TODO: 각 원소를 순회하면서
    ## 이미 seen에 있으면 duplicates에 추가
    ## 없으면 seen에 추가
    for n in nums:
        if n in seen:
            duplicates.add(n)
        else:
            seen.add(n)

    return list(duplicates)</code></pre>
