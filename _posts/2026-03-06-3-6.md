<p>느낀점</p>
<p>1. 남의 코드를 읽는게 나의 코드 작성보다 중요하다.</p>
<p>2. 나 혼자만의 일의 진행은 의미가 없다.</p>
<p>3. 내가 한일은 공유하고 나의 한계도 공개하자</p>
<p>4. 내가 작성한 코드의 검증(AI, 동료)이 필요하다</p>

<pre><code>from flask import Flask, render_template

app=Flask(__name__)

@app.route('/')
def home():
	name='LJH'
    return render_template('index.html',name=name)</code></pre>
